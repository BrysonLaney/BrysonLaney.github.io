Overview

Purpose

The purpose of this site is to help gamers discover new and interesting games recommended by other players. Users can browse suggestions and share their own with a link and short review.

Audience

This site is designed for video game enthusiasts of all ages who want to find hidden gems or share their favorite games with others in a community-driven space.

Dynamic Elements

JavaScript will dynamically display game suggestions on the Home page, validate form submissions to ensure a game name and link are provided, and update the list of suggestions without reloading. A like/upvote system will allow users to interact with posts, and filter/sort options will help visitors browse suggestions easily.

Content

Home Page

The Home page will display a welcome message such as: *“Welcome to Game Suggestions! Discover and share awesome games recommended by real players.”* It will include:

  • A featured list of user-submitted suggestions displayed as cards with game name, link, review, and like/upvote button.
  • Filter and sort options (e.g., by popularity or most recent).
  • A prominent link or button: *“Click here to submit your own suggestion!”* that takes users to the Submit page.

Submit Suggestion Page

The child page will contain:

  • A simple form with the following fields:
    • Game Name (*required*)
    • Game Link (*required*)
    • Your Review (*optional*)
  • Clear instructions: *“Please share only appropriate, genuine recommendations. Spam will be removed.”*
  • A Submit button that adds the new suggestion to the Home page dynamically.
